# landymaker 🚀

A professional Landing Page Builder built with Flutter and Supabase.

## Project Overview

"landymaker" allows users to build, customize, and deploy responsive landing pages in minutes using a modular block-based builder.

### Current Features

- **Advanced Builder Workspace**:
  - **Templates Tab**: Predefined starters (Store, Personal, Professional, TV Bar).
  - **Design Tab**: Real-time theme customization (5 preset palettes + custom hex selection).
  - **Content Tab**: Manage sections (Add, Reorder, Delete).
- **Comprehensive Section Library**:
  - `Hero`, `Features`, `Products` (4 layouts), `Lead Form`, `WhatsApp`, `QR Code`, `Social QR`, `Pricing`, `FAQ`, `Testimonials`, `Contact Info`, `Gallery`.
- **Responsive-First Design**:
  - Proportional scaling for all components.
  - adaptive padding and font sizes.
  - Zero-overflow architecture.

### Technical Stack

- **Flutter**: Clean architecture with BLoC (Cubit).
- **Supabase**: Handles persistent storage of page designs and user data.
- **SpecKit**: Used for project specification and task management.
